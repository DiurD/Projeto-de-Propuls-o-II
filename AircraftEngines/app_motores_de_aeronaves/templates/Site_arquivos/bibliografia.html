Bibliografia

Literaturas que foram utilizadas para o embasamento técnico deste site.

Elements of Gas Turbine Propulsion

Jack D. Mattingly

A metodologia de Mattingly sobre o cálculo de motores aeronáuticos, ideais e reais, é extremamente completa e de fácil entendimento. No AeroCalc, todas as fórmulas e parâmetros utilizados são de origem desta obra, sendo seu raciocínio utilizado para a geração de outras informações não disponíveis, como no cálculo das propriedades por seção de um motor.
Pode-se considerar esta a bibliografia principal da ferramenta, que deve ser consultada caso exista alguma dúvida quanto aos conteúdos utilizados aqui.

The Aerothermodynamics of Aircraft Gas Turbine Engines

Gordon C. Oates

Assim como na obra de Mattingly, Oates desenvolve modelos matemáticos para previsão de desempenho de motores. Entretanto, seu foco é no processo de otimização dos motores, buscando diferentes geometrias ou materiais que possam melhorar o desempenho encontrado.
Embora menos aplicado neste site, a bibliografia possui informações essenciais para o entendimento de modelos matemáticos para cálculo de motores aeronáuticos.

Desenvolvimento de um software online de fácil utilização para cálculo de parâmetros de motores aeronáuticos à reação sob condições de projeto e fora de projeto

Equipe do AeroCalc

Este trabalho realiza uma revisão e consolidação de estudos sobre análise de motores a jato, com base na metodologia de Mattingly. Um código Python foi desenvolvido e vinculado a um website interativo para facilitar o acesso dos usuários interessados.

ARP755A - Gas Turbine Engine Performance Station Identification and Nomenclature

SAE International

Esta norma é responsável por guiar a nomenclatura e identificação das seções dos motores aeronáuticos utilizados no AeroCalc.
